# BTC_price_predictor
Предиктор цен BTC на часовом таймфрейме.
Попытка сделать модель, которая сможет предсказывать с вероятностью свыше 50% рост цены BTC на закрытии следующей свечи без использования реккурентных нейросетей, Arima и каких-либо библиотек аналаза вр.рядов.
Так же пробую выполнить имитировать вывод модели в продакшн посредством внедрения её в ТГ бота с уведомлениями, подписками, БД пользователей.

Логика работы модели на примере:
Допустим, BTC закрыл свечу в 10:00 по UTC на уровне 5000$, по данным которые есть на текущий момент (цена закрытия этой свечи в 10:00 , её тени, скользящие средние цены по какому-то периоду и т.д) модель делает прогноз с какой вероятностью свеча в 11:00 закроется выше закрытия свечи в 10:00. 
Т.к эти вероятности будут плавать на каждом объекте (каждой свече), мы выбираем оптимальные параметры, при которых нас устроит итоговая точность на тестовой выборке и полнота.
